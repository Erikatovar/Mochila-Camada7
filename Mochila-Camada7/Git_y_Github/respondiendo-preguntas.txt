A: Git es un sistema de control de versiones distribuido. Esto significa que un clon local del proyecto es un repositorio de control de versiones completo. 
Estos repositorios locales plenamente funcionales permiten trabajar sin conexión o de forma remota fácilmente
Lo desarrolló originalmente Linus Torvalds.

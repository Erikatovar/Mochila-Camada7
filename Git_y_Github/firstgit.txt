¿Quién inventó el sistema de control de versión Git y por qué?  Linus Torvalds, el famoso creador del kernel del sistema operativo Linux. Algunos de los objetivos del nuevo sistema fueron los siguientes:

    Velocidad

    Diseño sencillo

    Gran soporte para desarrollo no lineal (miles de ramas paralelas)

    Completamente distribuido

    Capaz de manejar grandes proyectos (como el kernel de Linux) eficientemente (velocidad y tamaño de los datos)

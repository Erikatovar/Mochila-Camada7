Quien invento el sistema de control de version Git y por que?
Git fue creado por Chris Wanstrath, P. J Hyett, Tom Preston-Werner y Scott Chacon con el fin de que varios desarrolladores puedan trabajar al mismo tiempo.
A quien pertenece actualmente Github y por que?
Actualmente Github pertenece a Microsoft ya que lo compro por 7500 millones de dolares.
Hay otra forma que no sea la terminal para trabajar con Github?
Si, ademas de trabajar con la terminal desde nuestro computador, tenemos la opcion de trabajar desde el mismo sitio web.
